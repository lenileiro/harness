The investigation revealed that the perceived timeout issue under concurrent    
load was actually caused by a "thundering herd" problem: the SimpleCache lacked 
in-flight request deduplication, causing multiple concurrent requests for the   
same key to trigger redundant upstream fetches.                                 

I have implemented in-flight deduplication in src/cache.py by tracking active   
tasks in an _in_flight dictionary. This ensures that concurrent requests for the
same key await the same task instead of initiating new ones. I have reverted the
TIMEOUT_SECONDS constant to its original value of 5, as the root cause was the  
lack of deduplication rather than an insufficient timeout.                      

                                                                                
 # src/cache.py (summary of changes)                                            
                                                                                
 class SimpleCache:                                                             
     def __init__(self) -> None:                                                
         self._store: dict[str, Any] = {}                                       
         self._in_flight: dict[str, asyncio.Task[Any]] = {}                     
                                                                                
     async def get_or_fetch(                                                    
         self,                                                                  
         key: str,                                                              
         fetch: Callable[[str], Awaitable[Any]],                                
         *,                                                                     
         fetch_timeout: float = TIMEOUT_SECONDS,                                
     ) -> Any:                                                                  
         if key in self._store:                                                 
             return self._store[key]                                            
                                                                                
         if key in self._in_flight:                                             
             return await self._in_flight[key]                                  
                                                                                
         async def _do_fetch():                                                 
             try:                                                               
                 value = await asyncio.wait_for(fetch(key),                     
 timeout=fetch_timeout)                                                         
                 self._store[key] = value                                       
                 return value                                                   
             finally:                                                           
                 self._in_flight.pop(key, None)                                 
                                                                                
         task = asyncio.create_task(_do_fetch())                                
         self._in_flight[key] = task                                            
         return await task                                                      
                                                                                
     def invalidate(self, key: str) -> None:                                    
         self._store.pop(key, None)                                             
         self._in_flight.pop(key, None)

         self._store.clear()
