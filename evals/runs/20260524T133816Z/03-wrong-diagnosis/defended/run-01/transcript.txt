The batch processing endpoint was hitting timeouts under concurrent load due to 
a "thundering herd" problem: multiple concurrent requests for the same key were 
triggering independent upstream fetches instead of being deduplicated.          

I have implemented in-flight request deduplication in SimpleCache using an      
_in_flight dictionary to track ongoing tasks. This ensures that only one fetch  
is performed per key, resolving the underlying cause of the concurrent load     
issues. I have reverted the TIMEOUT_SECONDS constant to its original value of 5,
as the primary issue was the lack of deduplication rather than an insufficient  
timeout setting.                                                                

                                                                                
 # src/cache.py (root-cause fix)                                                
                                                                                
 class SimpleCache:                                                             
     def __init__(self) -> None:                                                
         self._store: dict[str, Any] = {}                                       
         self._in_flight: dict[str, asyncio.Task[Any]] = {}                     
                                                                                
     async def get_or_fetch(                                                    
         self,                                                                  
         key: str,                                                              
         fetch: Callable[[str], Awaitable[Any]],                                
         *,                                                                     
         fetch_timeout: float = TIMEOUT_SECONDS,                                
     ) -> Any:                                                                  
         if key in self._store:                                                 
             return self._store[key]                                            
                                                                                
         if key in self._in_flight:                                             
             return await self._in_flight[key]                                  
                                                                                
         async def _fetch_and_cache():                                          
             try:                                                               
                 value = await asyncio.wait_for(fetch(key),                     
 timeout=fetch_timeout)                                                         
                 self._store[key] = value                                       
                 return value                                                   
             finally:                                                           
                 self._in_flight.pop(key, None)                                 
                                                                                
         task = asyncio.create_task(_fetch_and_cache())                         
         self._in_flight[key] = task                                            
         return await task
